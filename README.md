# cucumber-bdd-testing

A sample BDD testing project using Cucumber, Selenium, Spring (Boot, AOP, Test). 

It uses the awesome [webdrivermanager](https://github.com/bonigarcia/webdrivermanager) to manage the drivers required by Selenium.

Aftear each scenario is executed, the software is able to save an evidence screenshot in the specified directory on application.yaml config file.

All the gherkin features are written in Portuguese.
